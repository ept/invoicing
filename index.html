invoicing by ept

Ruby invoicing framework gem

The Ruby invoicing framework provides tools, helpers and a structure for applications (particularly web apps) which need to generate invoices for customers. It builds on ActiveRecord and is particularly suited for Rails applications, but could be used with other frameworks too.

Dependencies

ActiveRecord >= 2.1

Install

sudo gem install invoicing
